# Smartcar | GM API Wrapper

Simple Smartcar API wrapper for the GM API.

## Getting Started

### Clone the repository

```
git clone https://github.com/hhovsepi/smartcar-gm-wrapper.git
cd smartcar-gm-wrapper
```

### Install dependencies

```
npm install
```

### Running in development

```
npm run dev
```

### Running in production

```
npm start
```

Runs on localhost:3000 by default but can be configured using the `PORT` environment variable.


### View documentation

Once you've started the server, navigate to /docs to view full documentation. 

```
http://localhost:3000/docs
```

### Running tests

```
npm test
```

### Linting
```
npm run lint

# fix issues
npm run lint:fix
```
